# innovate-irun

## Prerequisites
- Node.js (v18 or later)
- npm (v8 or later)

## Install dependencies:
- npm install

## Development mode
- npm run dev
## Production mode
- npm start

##API Endpoints
- GET /api/users - Get all users
- GET /api/users/:id - Get user by ID
- POST /api/users - Create a new user
